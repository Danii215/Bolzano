# Documentação do organismo Calculator

---

O organismo `Calculator` representa a calculadora, sendo construído das
moléculas `Table`, pra tabela que exibe os dados (resultados), e `CalcForm`,
que contém o formulário que recebe os dados do usuário para efetuar os
cálculos.

-   **Possui componente de Cliente: `CalcForm`**

## Props

Não aplicável.

import { Calculator } from "./Calculator";

    return <Calculator />;
}
