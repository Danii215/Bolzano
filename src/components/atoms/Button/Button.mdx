# Documentação do átomo Button

---

O componente `Button` é um botão simples, com a cor padrão do projeto. Pode ter
ou não uma setinha à direita. O intuito é ser um Call To Action.

-   **Componente de Servidor**

## Props

### text

-   **Tipo**: `string`
-   **Descrição**: O texto que aparece no botão.

### href

-   **Tipo**: `string`
-   **Descrição**: O link que o usuário é redirecionado ao clicar no botão.

### hasArrow

-   **Tipo**: `boolean`
-   **Descrição**: Define se deve existir uma setinha à direita.
