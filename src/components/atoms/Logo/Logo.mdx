# Documentação do átomo Logo

---

O componente `Logo` é utilizado para exibir o logotipo da aplicação, com a
opção de ser renderizado em dois tamanhos diferentes. Ao clicá-lo, o usuário é
redirecionado pra home.

-   **Componente de Servidor**

## Props

### isBig

-   **Tipo**: `boolean`
-   **Descrição**: Define se o logo deve ser exibido em um tamanho maior. Isso
    também muda a cor do texto para branco.

## Exemplo de uso

```tsx
import { Logo } from "./Logo";

function App() {
    return <Logo isBig />;
}
```
