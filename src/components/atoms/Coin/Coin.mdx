# Documentação do átomo Coin

---

O componente `Coin` é uma imagem qualquer, dentro de um círculo com a cor
padrão do projeto e sombreado abaixo.

Esse componente serve principalmente para
os banners introdutórios que existem em cada página.

-   **Componente de Servidor**

## Props

### title

-   **Tipo**: `string`
-   **Descrição**: Texto que aparecerá quando o usuário passa o mouse por cima
    da imagem, ou quando a imagem não carregar.

### image

-   **Tipo**: `StaticFile`
-   **Descrição**: Imagem dentro do círculo verde sombreado.
