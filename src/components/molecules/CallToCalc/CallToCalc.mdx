# Documentação da molécula CallToCalc

---

A molécula `CallToCalc` coloca um título (Content) e um botão de CTA (Call to
Action), para convidar o usuário à fazer alguma coisa. No caso padrão, à usar a
calculadora.

-   **Componente de Servidor**

## Props

Não aplicável.

}
