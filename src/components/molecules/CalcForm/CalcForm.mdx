# Documentação da molécula CalcForm

---

A molécula `CalcForm` reúne os 4 átomos de `TextInput` para obter do usuário o
que for necessário pra calcular.

A molécula também tem um input de submit para enviar a informação inserida pelo
usuário.

-   **Componente de Cliente**

## Props

Não aplicável.
