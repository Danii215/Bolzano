# Documentação da molécula Table

---

A molécula `Table` organiza headers e linhas em formato de tabela usando as
células `TableCell` respectivas.

-   **Componente de Servidor**

## Props

### header

-   **Tipo**: `Array<string>`
-   **Descrição**: Lista de nomes de cada coluna da tabela.

### rows

-   **Tipo**: `Array<Array<string>>`
-   **Descrição**: Lista de quantas linhas existirão na tabela. As arrays
    dentro dessa array define os valores exibidos nas células da respectiva linha.
